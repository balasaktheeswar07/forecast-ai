# 🚀 OLLAMA QUICK START GUIDE - See Your LLM Output NOW

## ⚡ 5-Minute Setup (Windows/Mac/Linux)

### Step 1: Install Ollama
Download from: **https://ollama.ai**

**After installation**, open terminal/command prompt and verify:
```bash
ollama --version
```

---

### Step 2: Pull Llama 3 Model (First time only - ~5 min)
```bash
ollama pull llama3
```

This downloads the trained Llama 3 model (~4GB). You only do this once.

---

### Step 3: Start Ollama Service
Keep this terminal open while using forecast-ai:
```bash
ollama serve
```

You should see:
```
2026/07/17 07:30:00 listening on 127.0.0.1:11434
```

✅ **Ollama is now running on http://localhost:11434**

---

### Step 4: Install Backend Dependencies
Open a **NEW terminal** (keep Ollama running in the other one):
```bash
cd backend
pip install -r requirements.txt
```

---

### Step 5: Start Forecast AI Backend
```bash
python run_backend.py
```

You should see:
```
Starting AI Forecasting Assistant backend on http://127.0.0.1:8000
INFO:     Uvicorn running on http://127.0.0.1:8000
```

---

### Step 6: Start React Frontend
Open a **THIRD terminal**:
```bash
cd frontend
npm install
npm run dev
```

Frontend starts at: **http://localhost:3000**

---

## 🛑 Troubleshooting

### **Ollama not connecting?**
```bash
# Check if Ollama is running
curl http://localhost:11434/api/tags

# If error, restart Ollama in terminal:
ollama serve
```

### **Model downloading slowly?**
- Ollama downloads ~4GB for Llama 3
- Wait it out or switch to smaller model: `ollama pull mistral`

### **Backend not finding Ollama?**
- Ensure Ollama is running: `ollama serve` in separate terminal
- Check port: `http://localhost:11434` should be accessible
- Logs will show: `✅ Ollama engine available`

### **Want faster inference?**
Switch to Mistral 7B (smaller, faster):
```bash
ollama pull mistral
# Then in code, model will auto-switch to available fastest model
```
